# Changelog

## 0.1.0

- Initial Movement Mode release.
- Added `SumMetric`, `CountMetric`, `RatioMetric`, and `AverageMetric`.
- Added additive decomposition for sum/count metrics.
- Added mix/rate/cross decomposition for ratio/average metrics.
- Added data profiler checks: row count, date coverage, null rate, duplicate grain.
- Added JSON, Markdown, and HTML reports.
- Added synthetic e-commerce demo.
- Added pytest suite and GitHub Actions CI.
